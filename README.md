# Sistema de Biblioteca

Aplicación de consola desarrollada en **C# (.NET)** que permite gestionar libros, usuarios y préstamos dentro de una biblioteca.

---

## Funcionalidades

El sistema incluye:

- Gestión de libros
  - Agregar libros
  - Listar libros

- Gestión de usuarios

- Agregar usuarios
- Listar usuarios

- Gestión de préstamos

- Registrar préstamos
- Listar préstamos

- Búsquedas y reportes
  - Buscar libros
  - Buscar usuarios
  - Ver reporte general

- ersistencia de datos
  - Guardar información en archivos `.txt`
  - Cargar información desde archivos

---

## 🛠️ Tecnologías utilizadas

- Lenguaje: **C#**
- Plataforma: **.NET (Console App)**
- Manejo de archivos: `System.IO`

---

## Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/denngoep/BibliotecaSintaxis.git
```

2. Entrar a la carpeta del proyecto:

```bash
cd BibliotecaSintaxis
```

3. Ejecutar la aplicación:

```bash
dotnet run
```

---

## Estructura del proyecto

- `Program.cs` → Contiene toda la lógica del sistema

## Notas

- Los datos se guardan localmente en archivos `.txt`
- Si los archivos no existen, se crean automáticamente al guardar
- El sistema funciona completamente desde consola
